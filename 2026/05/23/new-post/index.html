二.原题题面

You are given a room that can be represented by a n×m grid. There is a ball at position (i1,j1)
(the intersection of row i1and column j1), and it starts going diagonally in one of the four directions:

The ball is going down and right, denoted by DR; it means that after a step, the ball’s location goes from (i,j) to (i+1,j+1)
.
The ball is going down and left, denoted by DL
; it means that after a step, the ball’s location goes from (i,j) to (i+1,j−1)
.
The ball is going up and right, denoted by UR; it means that after a step, the ball’s location goes from (i,j)
to (i−1,j+1)
.
The ball is going up and left, denoted by UL
; it means that after a step, the ball’s location goes from (i,j)
to (i−1,j−1).
After each step, the ball maintains its direction unless it hits a wall (that is, the direction takes it out of the room’s bounds in the next step). In this case, the ball’s direction gets flipped along the axis of the wall; if the ball hits a corner, both directions get flipped. Any instance of this is called a bounce. The ball never stops moving.
Input
The first line contains a single integer t(1≤t≤1000) — the number of test cases.

The first line of each test case contains six integers and a string n,m,i1,j1,i2,j2,d
(2≤n,m≤25000; 1≤i1,i2≤n; 1≤j1,j2≤m; d∈{DR,DL,UR,UL}) — the dimensions of the grid, the starting coordinates of the ball, the coordinates of the final cell and the starting direction of the ball.

It is guaranteed that the sum of n⋅m
over all test cases does not exceed 5⋅$10^4$.
Output
For each test case, output a single integer — the number of bounces the ball does until it reaches cell (i2,j2)for the first time, or −1 if the ball never reaches the final cell.

三.题意+解题思路

1.题意：

一个小球从给定的(i1,j1)出发，按照d方向（DR：右下，DL：左下， UR：右上，UL：左上）开始弹射，碰到墙时沿墙的前进方向不变，垂直墙的前进方向相反；碰到角落时两个方向均改变，若能到(i2,j2)输出弹射次数，否则输出-1

2.解题思路

查找是否存在可行路径到达目标点，我们可以采用DFS的方法。每次计算出反弹的点的位置，并记录在tuple中表示已访问，同时判断目标点是否在前一个反弹点与当前反弹点所形成的线段上（可以通过叉积与是否在两点为对角线所形成的长方形内判断是否在线段内），若有，则找到，DFS结束，输出弹射次数，若无，继续DFS。如果发现反弹点已经记录在tuple中，则说明到不了目标点，DFS结束，输出-1。
