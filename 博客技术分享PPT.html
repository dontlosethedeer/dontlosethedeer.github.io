    • 安装 Hexo CLI
      npm install -g hexo-cli

      hexo init personal_blog && cd personal_blog
    • 安装主题
      git clone themes/next
    • 部署到 GitHub
      hexo deploy

    📊 功能实现：分类统计饼状图

    功能描述

    在分类页面显示饼状图，展示各分类文章数量


    技术方案

    • 使用 Chart.js 库
    • 手动配置分类数据
    • 响应式设计
    new Chart(ctx, { type: 'pie', data: { labels: categories.map(c => c.name), datasets: [{ data: categories.map(c => c.count), backgroundColor: ['#4A90D9'] }] } });

    ⚙️ 管理后台：hexo-admin

    功能特点

    • 可视化文章编辑器
    • 支持 Markdown 预览
    • 一键 Deploy 部署
    • 本地运行，无需联网

    使用方法

    • 安装
      npm install hexo-admin
    • 启动
      hexo server
    • 访问
      http://localhost:4000/admin
    • 登录
      用户名/密码: admin/admin123
